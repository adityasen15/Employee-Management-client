# Employee Management System (Client)

A frontend web application for managing employees including
add, update, delete, and view employee details.

## Features
- Add new employees
- Edit employee details
- Delete employees
- Responsive UI

## Tech Stack
- React
- JavaScript
- HTML
- CSS

## Getting Started
npm install
npm start
